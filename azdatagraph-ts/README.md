# Azdatagraph TS

This is a typescript library for creating graphs used to visualize SQL features.

## Building
To build the project, run the following command:
```bash
npm install
npm run build
```
For development, you can run the following command to watch for changes:
```bash
npm run watch
```

The project gets built into a single index.js under `dist` folder file which can be used in the browser or in a node environment.

## Installation

To use the library in your project

```bash
npm install github:Microsoft/azdataGraph#<version>
```

## Examples

To run the examples, run the following command:
```bash
npm run examples
```

This will start a local server and open the examples page in your default browser.
